# 目標 1

문제에는 없었던 실질적인 parking area를 구성한다.

# 目標 2

각 parking area는 스케줄러가 있어서 예약기능(선점기능)을 구현한다.

# 目標 3

예약을 실시할 시, 예약을 원하는 기간에 이미 앞선 예약으로 인해 선점된 경우 예약할 수 없는 자리임을 가시적으로 보여준다.
